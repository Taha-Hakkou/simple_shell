# SIMPLE_SHELL :

Simple UNIX command interpreter

## Project architecture :

. simple_shell/
	* README.md:
	* AUTHORS:
	* compile.sh:
	* hsh: 
	* Project files:
		** main.c:
		** shell.c:
		** shell.h:

## Requirements :

* Betty Style
* No memory leaks
* No more than 5 functions per file
* Header files should be include garded
* Use syscalls just in need !

## Compilation command :

gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o hsh
